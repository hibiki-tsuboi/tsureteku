「つれてく」は、お気に入りのぬいぐるみをAR（拡張現実）の力でスマホの中に取り込み、いつでもどこでも一緒にお出かけできるアプリです。

- **App Category**: エンターテインメント (Entertainment)
- **Deployment Target**: iOS 26.0+
- **プライバシーポリシー**: [https://hibiki-tsuboi.github.io/tsureteku/privacy-policy](https://hibiki-tsuboi.github.io/tsureteku/privacy-policy)

- **価格**: 無料 (Free)
- **年齢制限**: 4+ (なし)

### 概要 (Description)
```text
「つれてく」は、お気に入りのぬいぐるみをAR（拡張現実）の力でスマホの中に取り込み、いつでもどこでも一緒にお出かけできるアプリです。

旅行先での素敵な風景や、お気に入りのカフェでのひととき。
あなたの大切なキャラクターを現実の景色に呼び出して、一緒の思い出を写真に残しませんか？

【主な機能】
■ 簡単3Dスキャン（Object Capture）
最新のAR技術を使って、あなたのぬいぐるみを立体的にスキャン。カメラでぐるっと撮影するだけで、簡単にデジタルキャラクターとしてアプリに登録できます。

■ キャラクターライブラリ
スキャンしたキャラクターはライブラリに保存されます。いつでも一覧から選んで、その日のお出かけのお供を決めることができます。

■ ARカメラで記念撮影
登録したキャラクターを現実の風景に配置して、写真を撮影できます。大きさや位置も自由に調整可能。日常の景色が、もっと特別で楽しい空間に変わります。

【⚠️ 対応端末に関するご注意】
3Dスキャン（Object Capture）機能のご利用には、LiDARスキャナを搭載した端末が必要です。
（iPhone 12 Pro以降のProモデル、またはLiDAR搭載のiPad Pro等）
スキャン済みのキャラクターのAR表示や撮影機能については、LiDAR非搭載の端末でもご利用いただけます。

【こんな人におすすめ！】
・ぬいぐるみやキャラクターとお出かけ（ぬい撮り）をするのが好きな方
・推し活をもっと楽しく、もっと手軽にしたい方
・旅行やカフェ巡りの思い出を、自分らしく残したい方

お気に入りのあの子を、今日からいろんな場所へ「つれてく」体験をお楽しみください！
```

## ビルドと開発

- **Developer Mode**: 実機（iPhone / iPad）でビルドする場合は、設定アプリの「プライバシーとセキュリティ」から「デベロッパーモード」を有効にしてください。
- **Archive**: リリース用のアーカイブを作成する際は、ターゲットを「Any iOS Device (arm64)」に設定して `Product` > `Archive` を実行してください。
